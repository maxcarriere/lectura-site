Cette page décrira la politique de **confidentialité** du site Lectura.

Compte tenu du périmètre actuel (site vitrine statique, sans comptes utilisateurs ni base de données) :

- aucune donnée sensible n’est collectée par le site lui‑même ;
- seules les données techniques minimales nécessaires à l’hébergement peuvent être traitées par le fournisseur (par exemple GitHub Pages).

Le texte détaillé (cookies, éventuels outils de mesure d’audience, formulaires, etc.) sera rédigé et validé avant la mise en ligne publique.
